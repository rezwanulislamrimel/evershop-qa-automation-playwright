# Evershop QA Automation with Playwright

This repository contains end-to-end UI automation tests for the Evershop demo application using Playwright.

## Purpose
The goal of this project is to practice real-world QA automation by applying:
- Page Object Model (POM)
- Structured test scenarios
- Manual-to-automation thinking
- Regression-friendly test design

## Tech Stack
- Playwright
- Js
- Node.js
- Page Object Model (POM)

## Application Under Test
Evershop Demo (E-commerce platform)
